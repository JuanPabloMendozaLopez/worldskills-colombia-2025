# Segundo modulo - Gestión Escolar y Control de Usuarios

Este módulo se centró en el desarrollo de un sistema de administración escolar para la Escuela de Habilidades TI de WorldSkills. El objetivo principal era crear una aplicación completa que permitiera gestionar la información de estudiantes, profesores y personal administrativo de manera organizada y segura.

La prueba exigía trabajar con archivos de datos externos (formato CSV) e importarlos automáticamente a una base de datos. Uno de los mayores retos fue la gestión de permisos, ya que la aplicación debía comportarse de forma distinta según quién iniciara sesión

## Desempeño Personal

La experiencia previa en la primera competencia fue clave para afrontar la notable complejidad de este proyecto.
